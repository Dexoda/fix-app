# Fix App — Мобильная система контроля сотрудников

Мобильная веб-система (mobile-first) для контроля содержимого сумок сотрудников при выходе из магазина.

## Описание

Fix App позволяет:
- **Администраторам магазинов** — снимать фото/видео сотрудников или отмечать "Без сумки"
- **Директорам по безопасности** — просматривать и проверять записи с фильтрацией
- **Супер-администраторам** — управлять магазинами, сотрудниками и пользователями

## Технический стек

- **Backend**: Python 3.11 / FastAPI / SQLAlchemy / Alembic / Uvicorn
- **Frontend**: Vue 3 + Vite + Tailwind CSS + Pinia
- **БД**: PostgreSQL 15
- **Авторизация**: JWT (email + пароль)
- **Контейнеризация**: Docker + Docker Compose

## Требования

- Docker 20.10+
- Docker Compose 2.0+

## Быстрый старт

```bash
git clone <repo-url>
cd fix-app
docker-compose up --build
```

Приложение будет доступно по адресу: **http://localhost**

API документация (Swagger): **http://localhost/api/docs**

## Данные для входа по умолчанию

| Email | Пароль | Роль |
|-------|--------|------|
| admin@fixapp.local | admin123 | Супер-администратор |

## Роли пользователей

### Супер-админ (`super_admin`)
- CRUD магазинов, сотрудников, пользователей
- Просмотр логов
- Доступ ко всем данным

### Администратор магазина (`shop_admin`)
- Работа только со своим магазином
- Фото/видео/без сумки для сотрудников

### Директор по безопасности (`director`)
- Просмотр всех записей с фильтрами
- Отметка записей как "Проверено"

## API документация

Swagger UI доступен по адресу: `/api/docs`

Основные эндпоинты:
- `POST /api/v1/auth/login` — вход
- `POST /api/v1/auth/register` — регистрация
- `GET /api/v1/employees` — список сотрудников
- `POST /api/v1/records/upload` — загрузка фото/видео
- `POST /api/v1/records/no-bag` — отметка без сумки
- `GET /api/v1/records` — список записей
- `GET /api/v1/shops` — список магазинов
